# Unemployment Analysis with Python

This project is completed as part of the CodeAlpha Data Science Internship.

## Objective
To analyze unemployment trends in India and study the impact of Covid-19
using real-world data.

## Analysis Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Pre-Covid vs During-Covid comparison
- Monthly and yearly trend analysis
- Regional unemployment comparison
- Data visualization using plots

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Output
The analysis highlights unemployment trends, regional disparities,
seasonal patterns, and the significant impact of Covid-19.

Task completed successfully.
